# Controls: 
Space - Cycle (Generation)  
1 - Dead  
2 - Wire  
3 - Head  
4 - Tail  

A - Toggle Auto  
< - Decrease Auto Speed  
\> - Increase Auto Speed  

Alt-1 - Clear Screen Dead  
Alt-2 - Clear Screen Wire  
Alt-3 - Clear Screen Head  
Alt-4 - Clear Screen Tail  

Control-S - Save Map  
Control-O - Open Map
