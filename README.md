# Controls: 
Space - Cycle (Generation)  
1 - Dead  
2 - Wire  
3 - Head  
4 - Tail  
A - Toggle Auto  
< - Decrease Auto Speed  
\> - Increase Auto Speed  
Alt-C - Clear Screen
